## Overview

Morphine is the prototype opioid analgesic used for severe pain. It also causes significant venodilation, making it useful in acute pulmonary edema and MI, though this use is now questioned.

## NCLEX High-Yield Points

> ⚠️ MORPHINE: Hold if RR < 12/min. ANTIDOTE = NALOXONE (Narcan). All opioids cause constipation — start stool softeners immediately. Monitor respiratory rate and sedation. Tolerance develops with chronic use.

## Drug Class Memory Aid

**'Opioid triad: Analgesia + Sedation + Constipation'** — respiratory depression = most dangerous side effect
