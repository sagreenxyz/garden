## Overview

Buprenorphine is a partial opioid agonist with a ceiling effect on respiratory depression, making it safer than full agonists. It is a first-line treatment for opioid use disorder.

## NCLEX High-Yield Points

> ⚠️ , Buprenorphine: sublingual/buccal only — do NOT swallow. Precipitates withdrawal if started too soon (must wait for COWS > 8-12). Suboxone injected = naloxone becomes active = precipitated withdrawal. Ceiling effect on respiratory depression.

## Drug Class Memory Aid

**'Buprenorphine = partial agonist with ceiling'** — safer respiratory profile; sublingual, not swallowed
