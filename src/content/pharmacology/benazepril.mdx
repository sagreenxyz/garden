## Overview

Benazepril is a prodrug ACE inhibitor used for hypertension. Like all ACE inhibitors, it can cause dry cough, hyperkalemia, and angioedema.

## NCLEX High-Yield Points

> ⚠️ All ACE inhibitors: monitor for angioedema, cough, hyperkalemia, and renal function. Avoid in pregnancy.

## Drug Class Memory Aid

**ACE Inhibitors end in '-pril'** — all share same class effects
